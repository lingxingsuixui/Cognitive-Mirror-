# 🧠 Cognitive Mirror

A Multi-Agent Long Context Decision System

## 🚀 Overview
Cognitive Mirror is an AI system designed to move beyond simple question-answering into long-term cognitive modeling and decision simulation.

Instead of responding to a single query, the system:
- Builds a persistent user model using long-context memory
- Uses multiple agents to generate and evaluate decisions
- Simulates future outcomes for different choices
- Outputs structured decision trees

## 🧩 Architecture

User Memory (Long Context)
        ↓
Retrieval System
        ↓
Multi-Agent System (Competition + Voting)
        ↓
Critic (Risk Analysis)
        ↓
Simulator (Future Prediction)
        ↓
Decision Tree Output

## 🤖 Multi-Agent Roles

- Strategist A / B → Generate candidate decisions
- Voting Module → Select best option
- Critic → Identify risks
- Simulator → Predict outcomes
